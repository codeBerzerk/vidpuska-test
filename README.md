# Сервіс пошуку турів

Тестове завдання на розробку клієнтської частини сервісу пошуку турів.

## Технології

- **React 19** - бібліотека для побудови користувацьких інтерфейсів
- **TypeScript** - типізована надмножина JavaScript
- **Vite** - інструмент збірки та розробки
- **@floating-ui/react-dom** - утиліта для позиціонування випадаючих меню

## Встановлення та запуск

### Передумови

- Node.js (версія 18 або вище)
- npm або yarn

### Встановлення залежностей

```bash
npm install
```

### Запуск у режимі розробки

```bash
npm run dev
```

Застосунок буде доступний за адресою `http://localhost:5173`

### Збірка для продакшну

```bash
npm run build
```

### Попередній перегляд збірки

```bash
npm run preview
```

## Структура проєкту

```
src/
├── components/          # React компоненти
│   ├── Icons.tsx       # Компоненти іконок (країна, місто, готель)
│   ├── SearchInput.tsx # Компонент інпуту з випадаючим меню
│   ├── SearchInput.css # Стилі для SearchInput
│   ├── TourSearchForm.tsx # Головна форма пошуку
│   └── TourSearchForm.css  # Стилі для форми
├── types/              # TypeScript типи
│   └── api.ts         # Типи для даних API
├── utils/             # Утиліти
│   └── api.ts         # Обгортка над api.js
├── App.tsx            # Головний компонент
├── App.css            # Глобальні стилі
├── main.tsx           # Точка входу
└── index.css          # Базові стилі
```
